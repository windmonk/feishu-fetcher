# Feishu Fetcher - Quick Reference

## Commands

| Command | Description |
|---------|-------------|
| `--list` | List all files in Feishu media directory |
| `filename.html` | Fetch a specific file |
| `--latest` | Fetch the newest file |

## Examples

```bash
# List files
python skills/feishu-fetcher/scripts/feishu_fetcher.py --list

# Fetch file
python skills/feishu-fetcher/scripts/feishu_fetcher.py file.html

# Fetch latest
python skills/feishu-fetcher/scripts/feishu_fetcher.py --latest
```
